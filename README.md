# 41-牧场管理系统

##### 技术栈: springBoot+Redis+Mysql+Vue

##### 功能点: 牧场场地管理用户  畜牧信息管理用户  牧场产量管理用户    管理员端



##### 管理员端: 

1.系统用户: 牧场场地管理用户  畜牧信息管理用户  牧场产量管理用户   可对这三个类型的用户进行增加和查看及删除

2.牧场类型管理: 进行增加和查看及删除

3.牧场场地管理: 进行增加和查看及删除

4.畜牧种类管理: 进行增加和查看及删除

5.畜牧信息管理: 进行增加和查看及删除  可点击按钮进出牧场 记录时间

6.畜牧记录管理: 可进行审核进出牧场的提交信息和 查看 和删除记录

7.牧场产量管理: 进行增加和查看及删除

8.权限列表: 对用户进行权限设置
